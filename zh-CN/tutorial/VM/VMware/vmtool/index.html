解决 Workstation 问题合集（1）

发表于 2022-03-24 更新于 2024-08-23 分类于 教程 ， Windows ， VMware ， Workstation 阅读次数： 本文字数： 982 阅读时长 ≈ 3 分钟
通过补丁的办法给部分 Windows 安装 VMware Workstation 的 tool 以及解决 tool 安装按钮为灰色

用前说明

实验环境（其他版本 VMware Workstation 一样可以）

VMware Workstation 16.2.3
Windows 7 SP1（必须附带 SP1，如果你没有可以在下方补丁包内找到补丁）/Windows ES 7
Windows Vista（必须附带 SP2，如果你没有可以在下方补丁包内找到补丁）
Windows 2000
Windows NT4
补丁

章节 I（安装 tool）

Windows Vista/7 没有装 SP 的看这里

Windows 7

在虚拟机中插入补丁

点击虚拟机设置

在虚拟机设置中选择 CD/DVD，然后点击右边的使用 ISO 映像文件，并点击“浏览”，然后选择 win7-ser2008(R2)解决 vm16 无法安装 tool 的补丁.iso

选择完成后点击“确认”并返回到虚拟机

安装补丁

插入补丁后点击 Windows 徽标键>计算机

双击打开 CD 驱动器

在里面选择与你系统合适的补丁版本并安装

Windows Home Server 2011 请安装
适于 Windows Server 2008 R2 x64 的 安全更新 (KB4474419).msu

等待一会，然后选择“是(Y)”

大约会在 30s 左右完成

完成后点击“立即重新启动”

安装 tool

重启后安装正常方式安装 tool 即可

Windows Vista

在虚拟机中插入镜像

点击虚拟机设置

在虚拟机设置中选择 CD/DVD，然后点击右边的使用 ISO 映像文件，并点击浏览，然后选择 vm15tool.iso

选择完成后点击“确认”并返回到虚拟机

安装 tool

插入 tool 后点击 Windows 徽标键>计算机

然后在资源管理器选择 tool 安装即可

Windows 2000

在虚拟机中插入补丁

点击虚拟机设置

在虚拟机设置中选择 CD/DVD，然后点击右边的使用 ISO 映像文件，并点击浏览，然后选择 win2000 无法安装 tool 补丁.iso

选择完成后点击“确认”并返回到虚拟机

安装补丁

补丁插入后会自动运行安装界面
（如果没有就自己在我的电脑里选择插入的补丁镜像即可）

点击“我同意”——“下一步”

完成后点击“完成”即可自动重启

安装 tool

重启后安装正常方式安装 tool 即可

Windows NT4

在虚拟机中插入补丁

点击虚拟机设置

在虚拟机设置中选择 CD/DVD，然后点击右边的使用 ISO 映像文件，并点击“浏览”，然后选择 Windows NT 4.0 SP6a.iso

选择完成后点击“确认”并返回到虚拟机

安装补丁

补丁插入后会自动运行安装界面
（如果没有就自己在我的电脑里选择插入的补丁镜像即可）
然后勾选第一个复选框，第二个可以取消，然后点击“安装(I)”

等待一下，安装程序完成后点击“重新启动(R)”

安装 tool

重启后安装正常方式安装 tool 即可

如果你在安装过程遇到以下几个弹窗，点击“YES”、“OK”和“好”即可

章节 II（如果你安装 tool 的按钮是灰色）

—–如果你知道安装路径可忽略这段—–
在桌面上右键 VMware Workstation 快捷方式，然后选择“属性”

然后复制起始位置这个路径（不包括引号）

然后回到虚拟机，点击虚拟机设置

在虚拟机设置中选择 CD/DVD，然后点击右边的使用 ISO 映像文件，并点击浏览
（如果没有 CD/DVD 则需要自己在设置下面添加一个
在弹出的窗口中点击上方的地址栏空白处并将路径粘贴进去，然后回车

向下滑动找到 windows.iso 并选择，然后点击“打开(O)”

然后点击虚拟机设置下方的确定按钮即可
插入补丁后点击 Windows 徽标键>计算机

双击 CD 驱动器即可打开安装程序

然后正常安装 tool 即可
