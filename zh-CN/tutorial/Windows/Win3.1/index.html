虚拟机安装 Windows 3.1

发表于 2022-05-15 更新于 2024-08-23 分类于 教程 ， Windows 阅读次数： 本文字数： 509 阅读时长 ≈ 2 分钟
用 MS-DOS 7.10 在 Workstation 在虚拟机里安装 Windows 3.1

用前说明

实验环境

Windows 3.1、引导文件、MS-DOS
（Windows 3.1 源自 ITELLYOU MSDN
如果你真的够懒的话可以直接下里面的 Windows 3.1.iso
你也可以自己创建一个 MS-DOS，教程在下文）
UltraISO（在资源列表里有）
VMware Workstation（其它虚拟机也可以，部分内容有所不同）

创建镜像

如果你选择直接下 Windows 3.1.iso 的话可以掠过该步骤
先将所有文件解压出来，选择路径，点击“Extract”

打开“UltraISO”，在里面创建一个名为 Win3x 的文件夹，并打开它

打开 windows.31 下的“SIMPCHIN”文件夹，把所有文件复制进去

点击上面的“启动(B)”——“加载引导文件”，选择 boot.bif 文件


然后改改镜像显示名称（可不做）
最后点击“保存”，随便起个名就好

安装 Windows 3.1

先创建一个 MS-DOS 的虚拟机（或下载一个）

然后启动它
启动后点击“虚拟机设置”
选择 CD/DVD，把已连接(C)勾上，选择使用 ISO 映像文件(M)，然后点击浏览，选择创建好的镜像

点击确定

返回虚拟机后输入以下命令（最好区分大小写，不一定是 A：，也可能是 D：或者其它盘符）

然后回车

这样就进入了 Windows 3.1 的安装界面
