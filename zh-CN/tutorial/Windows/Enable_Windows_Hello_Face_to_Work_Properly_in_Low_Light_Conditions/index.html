在暗光下正常使用 Windows Hello Face

发表于 2025-10-06 更新于 2025-10-15 分类于 教程 ， Windows 阅读次数： 本文字数： 240 阅读时长 ≈ 1 分钟
禁用摄像头解决在暗光下不能正常使用 Windows Hello Face 的问题

前言

与其说是技术，不如说是分享一种邪门的解决方法吧，谁让 Windows 的技术债这么多 ¯\_(ツ)_/¯

解决方法

按下 WIN + R 以打开“运行”，输入 devmgmt.msc 并回车以打开“设备管理器”
展开“照相机”部分，将除了 ... IR Camera 以外的摄像头禁用即可（Virtual Camera 可不禁用）

后记

当然坏处就是不能正常使用摄像头了，如果摄像头不怎么用的话禁用了也罢 ¯\_(ツ)_/¯
