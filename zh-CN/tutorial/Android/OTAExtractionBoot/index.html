从 OTA 包中提取 boot

发表于 2023-10-21 更新于 2024-08-23 分类于 教程 ， Android 阅读次数： 本文字数： 186 阅读时长 ≈ 1 分钟
通过 payload-dumper-go 提取 OTA 包中的 boot.img

有时，Pixel 出厂镜像还没 OTA 更新的快，但是 Magisk 又需要 boot.img 来修补
这时候我们就可以 直接用旧的修补过的 boot.img 来用 从 OTA 包中提取出我们要用到的 boot.img 了

提取 boot.img

从 OneDrive 中下载 payload-dumper-go 并将其解压到空白文件夹（64 位系统用 payload-dumper-go-64，32 位系统用 payload-dumper-go-32，如果不知道你的 Windows 是多少位请直接用 payload-dumper-go-32）
然后解压 OTA 包里的 payload.bin 文件到同目录后运行 Runme.bat

然后在脚本里输入 b 后按下回车

完成后按任意键退出，脚本会打开提取出文件的文件夹
